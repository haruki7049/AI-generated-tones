# AI-generated-tones

This repository's codes were generated by [claude.ai](https://claude.ai) to create some tones, piano, orgen, and brass.

I don't know the licenses for the codes and tones, so I cannot apply any licenses for this repository.

## Build

```bash
# Enter into nix-shell
nix-shell
nix develop --print-build-logs

# Build this repository as normal ziglang repository
zig build

# Then there are some wav files
afplay zig-out/share/piano-tone.wav # macOS
aplay zig-out/share/organ-tone.wav # Linux / ALSA
pw-play zig-out/share/brass-tone.wav # Linux / Pipewire
```
